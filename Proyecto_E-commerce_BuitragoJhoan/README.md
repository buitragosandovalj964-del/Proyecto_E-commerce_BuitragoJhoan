Bienvenido al repositorio oficial de **SEBXZ EXCLUSIVE**, una plataforma de e-commerce enfocada en el sector *High-End Streetwear*. Este proyecto combina un diseño minimalista de alta gama con una gestión administrativa robusta y eficiente.

---

## 🚀 Sobre el Proyecto
SEBXZ EXCLUSIVE no es solo una tienda online; es una experiencia digital diseñada para el mercado de lujo. La plataforma ofrece un flujo de compra fluido para el cliente y un panel de control avanzado para el administrador, permitiendo gestionar el inventario, categorías y pedidos en tiempo real.

## ✨ Características Principales

### 🛒 Interfaz de Cliente (Storefront)
* **Catálogo Premium:** Visualización optimizada de productos.
* **Diseño Responsivo:** Adaptabilidad total en móviles y escritorio.
* **Experiencia inmersiva:** Interfaz limpia sin distracciones.

### 🔐 Panel Administrativo (System Admin)
* **Gestión de Productos:** Creación, edición y eliminación de prendas.
* **Control de Categorías:** Organización estructurada del catálogo.
* **Dashboard de Pedidos:** Seguimiento completo de las ventas.
* **Seguridad:** Acceso protegido mediante credenciales.

## 🛠 Tecnologías Utilizadas

| Tecnología | Rol |
| :--- | :--- |
